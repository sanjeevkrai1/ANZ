ANZ-FX - Application Type: Maven (Console based Java Application)
Java Version: 1.8
Libraries Used: JUnit & Mockito.

==============================Running ANZ-FX Application====================================================
Step to running ANZ-FX Application given below :-

Build: %>mvn clean install

Starting ANZ-FX-Calculator : Launch main method from class "AnzFxCalculatorApplication.java"

Converting Currency: Enter input in the form of "<ccy1> <amount1> in <ccy2>". 

E.g. %>AUD 100.00 in USD

Output Format: AUD 100.00 = USD 83.71

====================================Step to running Test cases for ANZ-FX Calculator=========================

Executing Test Cases: %> mvn test

Exiting the ANZ-FX-Calculator: %> exit
